## Local Usage

Requires Ruby 3.x (installed via `ruby-install`) and Jekyll.

```shell
# Install Ruby 3.3.6
brew install chruby ruby-install
ruby-install ruby 3.3.6

# Install Jekyll and Bundler
~/.rubies/ruby-3.3.6/bin/gem install jekyll bundler

# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve --watch
```

Visit `http://127.0.0.1:4000`
