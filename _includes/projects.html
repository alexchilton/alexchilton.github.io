GRPO Fine-tuning for Wordle Reasoning

GitHub

Applies Group Relative Policy Optimisation (GRPO) — the same RL technique used to train DeepSeek-R1 — to fine-tune a small language model to play Wordle strategically. The model learns to reason about letter constraints and improve its guessing strategy through reinforcement signals rather than supervised labels. CAS NLP Module 4 project at the University of Bern.

Stack: Python · Hugging Face TRL · GRPO · LoRA · Transformers

Fruit & Vegetable Recognition with Recipe Generation

GitHub

End-to-end computer vision pipeline that classifies fruit and vegetables from images using a CNN with transfer learning, then calls the OpenAI API to suggest a recipe based on the identified produce. Demonstrates combining classical CV with generative AI in a practical application. CAS AML Modules 1 & 2 project at the University of Bern.

Stack: Python · PyTorch · Transfer Learning · OpenAI API · Jupyter

Protein Generation with Graph Variational Autoencoders

GitHub

Generative model for protein structures using Graph Attention Networks (GAT) and Variational Autoencoders (VAE). The model learns a latent representation of protein graphs and generates novel structures by sampling from the learned distribution. Explores the intersection of graph neural networks and generative modelling in computational biology. CAS AML Module 3 project at the University of Bern.

Stack: Python · PyTorch Geometric · GAT · VAE · Graph Neural Networks · Jupyter
